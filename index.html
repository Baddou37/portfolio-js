Services

Développement Web

Le développement Web est le codage ou la programmation qui permet de faire fonctionner un site Web, selon les exigences du propriétaire. Il traite principalement de l’aspect non conceptuel de la création de sites Web, qui comprend le codage et l’écriture du balisage.

Web Design

le webdesign consiste à utiliser différents types de contenu pour transmettre et mettre en valeur des informations. Texte, image, vidéo, animation…

Intégration Web

L'intégration web coniste à monter les pages d'un site en réunissant l'ensemble des éléments décidés par le chef de projet ou le webmaster : textes, images, liens hypertextes, tableaux, etc.
